Connect to Cloudflare
3 Steps to Go Live
GitHub → Cloudflare Worker → Cloudflare Pages
1
Create a GitHub Repo
GitHub stores your files. Cloudflare reads from GitHub to deploy automatically. You only need to do this once — after that, any file you update in GitHub auto-deploys.
Create GitHub Repo → Login to GitHub →
Repo name suggestion: hustle-app
After creating repo: upload hustle_app.html and hustle_worker.js to it using the Add File button on GitHub.com — no terminal needed.
2
Deploy the Cloudflare Worker
The Worker is a tiny 50-line file that connects HUSTLE$ to Claude API securely. Your Anthropic API key lives in the Worker — never in the HTML file.
☁ Create Worker → Get Anthropic Key →
IN CLOUDFLARE WORKERS DASHBOARD:
1. Click "Create Worker" → name it hustle-worker
2. Click "Edit Code" → paste contents of hustle_worker.js
3. Click "Save and Deploy"
4. Go to Settings → Variables → Add Variable
5. Name: ANTHROPIC_KEY · Type: Secret · Value: your key
6. Copy your Worker URL (looks like: hustle-worker.YOUR-NAME.workers.dev)
Paste Your Worker URL Here
3
Deploy App to Cloudflare Pages
This puts HUSTLE$ on the internet — accessible from any device including your iPhone. Connect your GitHub repo and Cloudflare handles the rest automatically.
☁ Connect GitHub to Pages →
IN CLOUDFLARE PAGES:
1. Click "Connect to Git" → authorize GitHub
2. Select your hustle-app repo
3. Framework preset: None
4. Build command: leave blank
5. Build output: leave blank
6. Click "Save and Deploy"
7. Your URL: hustle-app.pages.dev
After deploy — open hustle-app.pages.dev in Safari on iPhone → Share → Add to Home Screen → HUSTLE$ works like a native app
